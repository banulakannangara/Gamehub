# GameHub - Cyberpunk Arcade Gaming Platform

A complete arcade gaming website featuring 5 fully playable mini games.

## 🎮 Games
- Snake Game
- Tic Tac Toe
- Click Speed Test
- Guess the Number
- Memory Card Game

## 🚀 How to run
Open index.html in browser

## ⚡ Features
- Cyberpunk UI
- Score saving (localStorage)
- Smooth animations
- Fully responsive design
